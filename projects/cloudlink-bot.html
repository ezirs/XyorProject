Serverless Telegram Bot API Web Aktif

CloudLink Bot

Layanan micro-service serverless berbasis API yang berfungsi sebagai jembatan (proxy) untuk mengubah file, gambar, dokumen, atau video dari Telegram menjadi tautan unduhan langsung (direct download link) secara instan — tanpa memerlukan database.

Kunjungi Website Coba di Telegram Dukung Proyek
Fitur Utama
⚡
All-in-One Architecture
Menggabungkan logika Bot Telegram (Webhook) dan URL Proxy di dalam satu infrastruktur serverless yang sama — tidak perlu deploy dua service terpisah.
🧠
Smart Content Handling
Otomatis mendeteksi tipe file. Gambar akan dirender langsung di browser, sedangkan dokumen dan arsip akan langsung diarahkan ke mode unduhan otomatis.
🌐
Dynamic Content Platform
Halaman utama (/) berfungsi sebagai landing page promosi statis yang sudah dioptimasi dengan SEO Meta Tags agar ramah mesin pencari.
💬
Interactive Monetization & Engagement
Respons bot menggunakan Inline Keyboard Button yang terintegrasi dengan platform Trakteer untuk mendukung skema donasi dari pengguna.
↩️
Smart Reply System
Setiap respons bot — baik link sukses maupun pesan kesalahan — dikirim dengan metode Reply yang mengikat pada pesan asli pengguna agar alur chat tetap rapi.
📏
Validasi Ukuran File
Dilengkapi pembatasan otomatis ukuran file hingga maksimal 20 MB per file demi menjaga efisiensi dan kestabilan layanan.
Keunggulan Sistem
💸
Zero Cost Infrastructure
Berjalan 100% di Free Tier Cloudflare Workers dengan batas 100.000 request/hari — tanpa biaya VPS sama sekali.
🔒
High Security
Token Bot Telegram terisolasi di sisi server (Cloudflare) dan tidak akan bocor ke publik meskipun file diakses via URL kustom.
⚡
Edge Performance
Dijalankan di edge network global Cloudflare menggunakan V8 JavaScript Engine untuk latensi yang sangat rendah.
🔄
Stateless by Design
Tidak ada state atau sesi yang disimpan — setiap request independen, membuat sistem mudah di-scale secara horizontal.
